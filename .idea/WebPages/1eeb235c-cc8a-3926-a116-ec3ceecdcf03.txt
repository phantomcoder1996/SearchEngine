         <h2>2. Handler Mapping</h2> 
         <p>Define how web request (URL) maps to the Controller handlers.</p> 
         <ul> 
          <li><a href="http://www.mkyong.com/spring-mvc/spring-mvc-beannameurlhandlermapping-example/">BeanNameUrlHandlerMapping example</a><br> Maps the requested URL to the name of the cocntroller. </li> 
          <li><a href="http://www.mkyong.com/spring-mvc/spring-mvc-controllerclassnamehandlermapping-example/">ControllerClassNameHandlerMapping example</a><br> Uses convention to map the requested URL to Controller. </li> 
          <li><a href="http://www.mkyong.com/spring-mvc/spring-mvc-simpleurlhandlermapping-example/">SimpleUrlHandlerMapping example</a><br> Allow developer to specify the mapping of URL patterns and handler mappings explicitly. </li> 
          <li><a href="http://www.mkyong.com/spring-mvc/configure-the-handler-mapping-priority-in-spring-mvc/">Configure the handler mapping priority</a><br> If multiple view handler mappings are applied, you have to declare priority to avoid conflict issue. </li> 
         </ul>

         <h2>3. Controller</h2> 
         <p>Controller class to handle the web request.</p> 
         <ul> 
          <li><a href="http://www.mkyong.com/spring-mvc/spring-mvc-multiactioncontroller-example/">MultiActionController example</a><br> Group related actions into a single controller class. </li> 
          <li><a href="http://www.mkyong.com/spring-mvc/spring-mvc-multiactioncontroller-annotation-example/">MultiActionController annotation example</a><br> Multi-actions controller by using annotation, @RequestMapping, he best and easy solution. </li> 
          <li><a href="http://www.mkyong.com/spring-mvc/spring-mvc-propertiesmethodnameresolver-example/">PropertiesMethodNameResolver example</a><br> A flexible method name resolver for the MultiActionController controller class, it allow to define the mapping between requested URL and method name explicitly. </li> 
          <li><a href="http://www.mkyong.com/spring-mvc/spring-mvc-parametermethodnameresolver-example/">ParameterMethodNameResolver example</a><br> Another method name resolver for the MultiActionController controller class, it allow to map URL to method name via requested parameter name. </li> 
          <li><a href="http://www.mkyong.com/spring-mvc/spring-mvc-parameterizableviewcontroller-example/">ParameterizableViewController example</a><br> With ParameterizableViewController, you don’t need to hard code the view name in the controller class anymore, instead, you specify the view name through the ParameterizableViewController’s “viewName” property. </li> 
         </ul> 
         <h2>4. View Resolver</h2> 
         <p>Resolve “view name” that returned from the controller class to a physical view page or JSP page.</p> 
         <ul> 
          <li><a href="http://www.mkyong.com/spring-mvc/spring-mvc-internalresourceviewresolver-example/">InternalResourceViewResolver example</a><br> Adding a predefined prefix and suffix to the view name (prefix + view name + suffix), and generate the final view page URL. </li> 
          <li><a href="http://www.mkyong.com/spring-mvc/spring-mvc-xmlviewresolver-example/">XmlViewResolver example</a><br> Puts view beans in XML file. </li> 
          <li><a href="http://www.mkyong.com/spring-mvc/spring-mvc-resourcebundleviewresolver-example/">ResourceBundleViewResolver example</a><br> Puts view beans in “.properties” file. </li> 
          <li><a href="http://www.mkyong.com/spring-mvc/configure-multiple-view-resolvers-priority-in-spring-mvc/">Configure multiple view resolvers priority</a><br> If multiple view resolver strategies are applied, you have to declare priority to avoid conflict issue. </li> 
         </ul> 
         <h2>5. Form Handling</h2> 
         <p>Form handling in Spring MVC.</p> 
         <ul> 
          <li><a href="http://www.mkyong.com/spring-mvc/spring-mvc-form-handling-example/">Form handling example</a><br> Form handling in Spring MVC, XML based version. </li> 
          <li><a href="http://www.mkyong.com/spring-mvc/spring-mvc-form-handling-annotation-example/">Form handling annotation example</a><br> Form handling in Spring MVC, annotation version. </li> 
          <li><a href="http://www.mkyong.com/spring-mvc/handling-duplicate-form-submission-in-spring-mvc/">Handling duplicate form submission</a><br> “Post/Redirect/Get” Design Pattern is the common solution for this duplicated form submission problem. </li> 
          <li><a href="http://www.mkyong.com/spring-mvc/spring-mvc-redirectview-example/">RedirectView example</a><br> A view which redirect to another absolute, context relative, or current request relative URL. </li> 
          <li><a href="http://www.mkyong.com/spring-mvc/spring-mvc-handling-multipage-forms-with-abstractwizardformcontroller/">Handling multipage / wizard forms</a><br> How to handle multipage forms or wizard form. </li> 
         </ul> 
         <h2>6. Spring’s Form Tag Library</h2> 
         <p>Render HTML form’s components via Spring’s form tag.</p> 
         <ul> 
          <li><a href="http://www.mkyong.com/spring-mvc/spring-mvc-textbox-example/">Textbox example</a><br> &lt;form:input /&gt; tag, render a HTML textbox field. </li> 
          <li><a href="http://www.mkyong.com/spring-mvc/spring-mvc-password-example/">Password example</a><br> &lt;form:password /&gt; tag, render a HTML password field. </li> 
          <li><a href="http://www.mkyong.com/spring-mvc/spring-mvc-textarea-example/">Textarea example</a><br> &lt;form:textarea /&gt; tag, render a HTML textarea field. </li> 
          <li><a href="http://www.mkyong.com/spring-mvc/spring-mvc-checkbox-and-checkboxes-example/">Checkbox and Checkboxes example</a><br> &lt;form:checkbox /&gt;, &lt;form:checkboxes /&gt; tags, render a single or multiple HTML check boxes. </li> 
          <li><a href="http://www.mkyong.com/spring-mvc/spring-mvc-radiobutton-and-radiobuttons-example/">Radiobutton and Radiobuttons example</a><br> &lt;form:radiobutton /&gt;, &lt;form:radiobuttons /&gt; tags, render a single or multiple HTML radio buttons. </li> 
          <li><a href="http://www.mkyong.com/spring-mvc/spring-mvc-dropdown-box-example/">Dropdown and list box example</a><br> &lt;form:select /&gt;, &lt;form:option /&gt; and &lt;form:options /&gt; tags, render a HTML dropdown box, multiple select box and listbox. </li> 
          <li><a href="http://www.mkyong.com/spring-mvc/spring-mvc-hidden-value-example/">Hidden value example</a><br> &lt;form:hidden /&gt; tags, render a HTML hidden value field. </li> 
          <li><a href="http://www.mkyong.com/spring-mvc/spring-mvc-form-errors-tag-example/">Form errors tag example</a><br> &lt;form:errors /&gt; tag , render those form components error messages. </li> 
          <li><a href="http://www.mkyong.com/spring-mvc/spring-mvc-file-upload-example/">File upload example</a><br> File upload handling via Spring MultipartResolver. </li> 
         </ul> 
         <h2>7. Integrating Spring MVC with other frameworks</h2> 
         <p>Integrate Spring MVC with others.</p>
