        <h1>Spring Security Tutorial</h1> 
        <div class="post-meta"> 
         <div class="post-meta">
          By 
          <a href="http://www.mkyong.com/author/mkyong/" title="mkyong class=" author" rel="author">mkyong</a> |

         <div class="pic"> 
          <img src="http://www.mkyong.com/wp-content/uploads/2011/08/spring-security-logo.png" alt="spring security tutorials" width="435" height="193" class="alignnone size-full wp-image-13351" srcset="http://www.mkyong.com/wp-content/uploads/2011/08/spring-security-logo.png 435w, http://www.mkyong.com/wp-content/uploads/2011/08/spring-security-logo-300x133.png 300w" sizes="(max-width: 435px) 100vw, 435px"> 
         </div> 
         <p><a href="http://projects.spring.io/spring-security/" target="_blank">Spring Security</a>, is a flexible and powerful authentication and access control framework to secure Spring-based Java web application.</p> 
         <p>Spring version to use in this tutorials :</p> 
         <ol> 
          <li>Spring 3.2.8.RELEASE</li> 
          <li>Spring Security 3.2.3.RELEASE</li> 
         </ol> 
         <div class="note"> 
          <strong>Note</strong>
          <br> Try this 
          <a href="http://www.mkyong.com/spring-boot/spring-boot-spring-security-thymeleaf-example/" target="_blank">Spring Boot + Spring Security + Thymeleaf example</a> 
         </div> 
         <h2>1. Spring Security Examples</h2> 
         <p>Examples to show you how to secure your web application with Spring Security.</p> 
         <ul> 
          <li><a href="http://www.mkyong.com/spring-security/spring-security-hello-world-example/">Spring Security Hello World XML Example</a><br> Spring MVC + Spring Security XML-based project, using the default login form. </li> 
          <li><a href="http://www.mkyong.com/spring-security/spring-security-hello-world-annotation-example/">Spring Security Hello World Annotation Example</a><br> Spring MVC + Spring Security annotations-based project, using the default login form. </li> 
          <li><a href="http://www.mkyong.com/spring-security/spring-security-form-login-example/">Spring Security Custom Login Form XML Example</a><br> Spring MVC + Spring Security XML-based project, custom login form, logout function, CSRF protection and in-memory authentication. </li> 
          <li><a href="http://www.mkyong.com/spring-security/spring-security-custom-login-form-annotation-example/">Spring Security Custom Login Form Annotation Example</a><br> Spring MVC + Spring Security annotations-based project, custom login form, logout function, CSRF protection and in-memory authentication. </li> 
          <li><a href="http://www.mkyong.com/spring-security/spring-security-form-login-using-database/">Spring Security Form Login Using Database – XML and Annotation Example</a><br> Database authentication, Spring Security, JSP taglibs, JDBC, customizes 403 access denied page and etc, both in XML and annotations. </li> 
          <li><a href="http://www.mkyong.com/spring-security/spring-security-limit-login-attempts-example/">Spring Security : Limit Login Attempts – XML and Annotation Example</a><br> Lock user accounts if a user tried 3 invalid login attempts. </li> 
          <li><a href="http://www.mkyong.com/spring-security/spring-security-remember-me-example/">Spring Security Remember Me Example</a><br> Remember me “Persistent Token Approach” example. Extra : user login from remember me cookie is unable to perform the update operation. </li> 
          <li><a href="http://www.mkyong.com/spring-security/spring-security-password-hashing-example/">Spring Security password hashing example</a><br> Password encoder with BCrypt algorithm. </li> 
          <li><a href="http://www.mkyong.com/spring-security/spring-security-hibernate-xml-example/">Spring Security + Hibernate XML Example</a><br> Using Hibernate to load users for database authentication. </li> 
          <li><a href="http://www.mkyong.com/spring-security/spring-security-hibernate-annotation-example/">Spring Security + Hibernate Annotation Example</a><br> Using Hibernate to load users for database authentication. </li> 
         </ul>

         <h2>2. FAQs</h2> 
         <p>Some commonly asked questions in Spring Security.</p> 
         <ul> 
          <li><a href="http://www.mkyong.com/spring-security/customize-http-403-access-denied-page-in-spring-security/">Customize 403 access denied page</a><br> Like topic, shows you how to customize a 403 access denied page, using the error-page attribute and custom handler. </li> 
          <li><a href="http://www.mkyong.com/spring-security/spring-security-check-if-user-is-from-remember-me-cookie/">Check if user is from remember me cookie</a><br> If authentication == RememberMeAuthenticationToken </li> 
          <li><a href="http://www.mkyong.com/spring-security/spring-security-encoded-password-does-not-look-like-bcrypt/">Encoded password does not look like BCrypt</a><br> Length of “password” is not enough to store the bcrypt hashing value. </li> 
          <li><a href="http://www.mkyong.com/spring-security/get-current-logged-in-username-in-spring-security/">How to get current login username in Spring Security</a><br> 3 ways to get current login username in Spring Security. </li> 
          <li><a href="http://www.mkyong.com/spring-security/classnotfoundexception-defaultsavedrequest/">ClassNotFoundException : org.springframework.security.web.savedrequest.DefaultSavedRequest</a> </li>
