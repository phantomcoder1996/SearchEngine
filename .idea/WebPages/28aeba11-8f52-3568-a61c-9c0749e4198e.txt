        <h1>Spring Tutorial</h1> 
        <div class="post-meta"> 
         <div class="post-meta">

         <p>The <a href="http://www.springsource.org/" target="_blank">Spring framework</a> , created by Rod Johnson, is an extremely powerful Inversion of control(IoC) framework to help decouple your project components’ dependencies.</p> 
         <p>In this series of tutorials, it’s provides many step by step examples and explanations on using the Spring framework.</p> 
         <div class="note"> 
          <strong>New Spring 3.0 Tutorials (23/06/2011)</strong>
          <br> Added many Spring 3.0 tutorials on using Spring EL, JavaConfig, AspectJ and Spring Object/XML mapping (oxm). For what’s new in Spring 3.0, you can refer to this 
          <a href="http://static.springsource.org/spring/docs/3.0.x/spring-framework-reference/html/new-in-3.html" target="_blank">official Spring 3.0 references</a>. 
         </div>

         <p>Quick start to understand the basic of Spring framework development.</p> 
         <ul> 
          <li><a href="http://www.mkyong.com/spring/quick-start-maven-spring-example/">Spring hello world example </a><br> Maven + Spring 2.5.6 hello world example.</li> 
          <li><a href="http://www.mkyong.com/spring3/spring-3-hello-world-example/">Spring 3.0 hello world example</a> <span><strong>(Spring 3.0)</strong></span><br> Maven + Spring 3.0 hello world example, what are needed in new Spring 3.0 development.</li> 
          <li><a href="http://www.mkyong.com/spring/spring-loosely-coupled-example/">Spring loosely coupled example</a><br> An example to show demonstrate how Spring make components loosely coupled. </li> 
         </ul>

         <p>Spring 3.0 supported JavaConfig, now you can use annotations to do the configuration in Spring. </p> 
         <ul> 
          <li><a href="http://www.mkyong.com/spring3/spring-3-javaconfig-example/">Spring 3 JavaConfig examples</a><br> Demonstrate the use of @Configuration and @Bean to define bean in Spring</li> 
          <li><a href="http://www.mkyong.com/spring3/spring-3-javaconfig-import-example/">Spring 3 JavaConfig @Import example</a><br> Demonstrate the use of @Import to organize beans in modular.</li>

         <h2>Spring Dependency Injection (DI)</h2> 
         <p>How Spring to do dependency Injection (DI) to manage object dependencies.</p> 
         <ul> 
          <li><a href="http://www.mkyong.com/spring/spring-dependency-injection-di/">Spring Dependency Injection (DI)</a><br> How Spring applies the Dependency Injection (DI) design pattern via Setter Injection and Constructor Injection.</li> 
          <li><a href="http://www.mkyong.com/spring/spring-di-via-setter-method/">Spring DI via setter method</a><br> Dependency injection a bean via setter method.</li> 
          <li><a href="http://www.mkyong.com/spring/spring-di-via-constructor/">Spring DI via constructor</a><br> Dependency injection a bean via constructor.</li> 
          <li><a href="http://www.mkyong.com/spring/constructor-injection-type-ambiguities-in-spring/">Constructor injection type ambiguities in Spring</a><br> The constructor injection argument type ambiguity issue is always happening in a bean which contains multiple constructor methods with many arguments.</li> 
         </ul> 
         <h2>Bean Basic</h2> 
         <p>All the classes you need to use in Spring Ioc container are considered “bean”, and declared in a Spring bean configuration file or via annotation.</p> 
         <ul> 
          <li><a href="http://www.mkyong.com/spring/spring-bean-reference-example/">Spring bean reference example</a><br> How beans access to each other by specify the bean references in the same or different bean configuration file.</li> 
          <li><a href="http://www.mkyong.com/spring/how-to-define-bean-properties-in-spring/">Inject value into bean properties in Spring</a><br> Three ways to inject value into bean properties.</li> 
          <li><a href="http://www.mkyong.com/spring/load-multiple-spring-bean-configuration-file/">Load multiple Spring bean configuration file</a><br> Developers always categorize different bean configuration files in different modules folder, here’s a tip to show you how to load multiple Spring bean configuration files.</li> 
          <li><a href="http://www.mkyong.com/spring/spring-inner-bean-examples/">Spring inner bean examples</a><br> Whenever a bean is used for one particular property only, it’s always advised to declare it as an inner bean.</li> 
          <li><a href="http://www.mkyong.com/spring/spring-bean-scopes-examples/">Spring bean scopes examples</a><br> Bean scope is used to decide which type of bean instance should be return from the Spring container back to the caller.</li> 
          <li><a href="http://www.mkyong.com/spring/spring-collections-list-set-map-and-properties-example/">Spring Collections (List, Set, Map, and Properties) example</a><br> Example to inject values into collections type (List, Set, Map, and Properties).</li> 
          <li><a href="http://www.mkyong.com/spring/spring-listfactorybean-example/">ListFactoryBean example</a><br> Create a concrete List collection class (ArrayList and LinkedList), and inject it into bean property.</li> 
          <li><a href="http://www.mkyong.com/spring/spring-setfactorybean-example/">SetFactoryBean example</a><br> Create a concrete Set collection class (HashSet and TreeSet), and inject it into bean property.</li> 
          <li><a href="http://www.mkyong.com/spring/spring-mapfactorybean-example/">MapFactoryBean example</a><br> Create a concrete Map collection class (HashMap and TreeMap), and inject it into bean property.</li> 
          <li><a href="http://www.mkyong.com/spring/spring-how-to-pass-a-date-into-bean-property-customdateeditor/">Spring inject Date into bean property – CustomDateEditor</a><br> Normally, Spring is accepting date variable, here’s a tip to use CustomDateEditor to work around it.</li> 
          <li><a href="http://www.mkyong.com/spring/spring-propertyplaceholderconfigurer-example/">Spring PropertyPlaceholderConfigurer example</a><br> Externalize the deployment details into a properties file, and access from a bean configuration file via a special format – ${variable}.</li> 
          <li><a href=" http://www.mkyong.com/spring/spring-bean-configuration-inheritance/">Spring bean configuration inheritance</a><br> Inheritance is very useful for a bean to share common values, properties or configuration.</li> 
          <li><a href="http://www.mkyong.com/spring/spring-properties-dependency-checking/">Spring dependency checking</a><br> Spring comes with 4 dependency checking modes to make sure the required properties have been set in bean.</li> 
          <li><a href="http://www.mkyong.com/spring/spring-dependency-checking-with-required-annotation/">Spring dependency checking with @Required Annotation</a><br> Dependency checking in annotation mode.</li> 
          <li><a href="http://www.mkyong.com/spring/spring-define-custom-required-style-annotation/">Custom @Required-style annotation</a><br> Create a custom @Required-style annotation ,which is equivalent to @Required annotation.</li> 
          <li><a href="http://www.mkyong.com/spring/spring-initializingbean-and-disposablebean-example/">Bean InitializingBean and DisposableBean example</a><br> Perform certain actions upon bean initialization and destruction. (interface)</li> 
          <li><a href="http://www.mkyong.com/spring/spring-init-method-and-destroy-method-example/">Bean init-method and destroy-method example</a><br> Perform certain actions upon bean initialization and destruction. (XML)</li> 
          <li><a href="http://www.mkyong.com/spring/spring-postconstruct-and-predestroy-example/">Bean @PostConstruct and @PreDestroy example</a><br> Perform certain actions upon bean initialization and destruction. (Annotation)</li> 
         </ul> 
         <h2>Spring Expression Language (Spring 3.0)</h2> 
         <p>Spring 3.0 introduces features rich and powerful expression language known as Spring expression language, or Spring EL.</p> 
         <ul> 
          <li><a href="http://www.mkyong.com/spring3/spring-el-hello-world-example/">Spring EL hello world example</a><br> Quick start to use the Spring expression language (EL).</li> 
          <li><a href="http://www.mkyong.com/spring3/spring-el-bean-reference-example/">Spring EL bean reference example</a><br> Reference bean, bean property using a dot (.) symbol.</li> 
          <li><a href="http://www.mkyong.com/spring3/spring-el-method-invocation-example/">Spring EL method invocation example</a><br> Call bean method in directly.</li> 
          <li><a href="http://www.mkyong.com/spring3/spring-el-operators-example/">Spring EL operators example</a><br> Spring EL supports most of the standard relational, logical and mathematical operators.</li> 
          <li><a href="http://www.mkyong.com/spring3/spring-el-ternary-operator-if-then-else-example/">Spring EL ternary operator (if-then-else) example</a><br> Conditional check , if else then.</li> 
          <li><a href="http://www.mkyong.com/spring3/spring-el-lists-maps-example/">Spring EL Arrays, Lists, Maps example</a><br> Works with Map and List.</li> 
          <li><a href="http://www.mkyong.com/spring3/spring-el-regular-expression-example/">Spring EL regular expression example</a><br> The regular expression to evaluate the condition.</li> 
          <li><a href="http://www.mkyong.com/spring3/test-spring-el-with-expressionparser/">Test Spring EL with ExpressionParser</a><br> Show you how to test Spring El easily.</li> 
         </ul> 
         <h2>Spring Auto Component Scanning</h2> 
         <p>Spring is able to scan, detect and register your bean automatically.</p> 
         <ul> 
          <li><a href="http://www.mkyong.com/spring/spring-auto-scanning-components/">Spring Auto scanning components</a><br> Enable Spring to auto scan, detect and register your beans.</li> 
          <li><a href="http://www.mkyong.com/spring/spring-filtering-components-in-auto-scanning/">Spring Filter components in auto scanning</a><br> Example to filter certain components in auto scanning mode.</li> 
         </ul> 
         <h2>Spring AutoWiring Bean</h2> 
         <p>Spring ‘auto-wiring’ modes to wire or beans automatically, both in XML and annotation.</p> 
         <ul> 
          <li><a href="http://www.mkyong.com/spring/spring-auto-wiring-beans-in-xml/">Spring Auto-Wiring Beans</a><br> Summary of 5 types auto wiring modes in Spring.</li> 
          <li><a href="http://www.mkyong.com/spring/spring-autowiring-by-type/">Spring Autowiring by Type</a><br> If the data type of a bean is compatible with the data type of other bean property, auto wire it.</li> 
          <li><a href="http://www.mkyong.com/spring/spring-autowiring-by-name/">Spring Autowiring by Name</a><br> If the name of a bean is same as the name of another bean property, auto wire it.</li> 
          <li><a href="http://www.mkyong.com/spring/spring-autowiring-by-constructor/">Spring Autowiring by Constructor</a><br> Actually, it’s autowiring by Type in constructor argument.</li> 
          <li><a href="http://www.mkyong.com/spring/spring-autowiring-by-autodetect/">Spring Autowiring by AutoDetect</a><br> It means choosing “autowire by constructor” if default constructor is found, otherwise uses “autowire by type”.</li> 
          <li><a href="http://www.mkyong.com/spring/spring-auto-wiring-beans-with-autowired-annotation/">Spring Autowiring with @Autowired annotation</a><br> Examples to show how to define ‘auto-wiring’ modes in annotation.</li> 
          <li><a href="http://www.mkyong.com/spring/spring-autowiring-qualifier-example/">Spring Autowiring @Qualifier example</a><br> Example to identify which bean is qualified to autowired on a field.</li> 
         </ul> 
         <h2>Spring AOP (Aspect-oriented programming)</h2> 
         <p>The Spring AOP modularize cross-cutting concerns in aspects. Put it simply, an interceptor to intercept some methods.</p> 
         <ul> 
          <li><a href="http://www.mkyong.com/spring/spring-aop-examples-advice/">Spring AOP Example – Advice</a><br> Examples and explanations about different types of Spring’s advices.</li> 
          <li><a href="http://www.mkyong.com/spring/spring-aop-example-pointcut-advisor/">Spring AOP Example – Pointcut , Advisor</a><br> Examples and explanations about different types of Spring’s Pointcut and Advisor.</li> 
          <li><a href="http://www.mkyong.com/spring/spring-aop-interceptor-transaction-is-not-working/">Spring AOP interceptor sequence</a><br> The sequence of the AOP interceptor will affect the functionality.</li> 
          <li><a href="http://www.mkyong.com/spring/spring-auto-proxy-creator-example/">Auto proxy creator example</a><br> An auto proxy creator example to create proxy object for your beans automatically, useful to avoid creating many duplicated proxy objects.</li> 
         </ul> 
         <h2>Spring AOP + AspectJ framework</h2> 
         <p>AspectJ supported since Spring 2.0, more flexible and powerful. However, this example is demonstrated in Spring 3.0. </p> 
         <ul> 
          <li><a href="http://www.mkyong.com/spring3/spring-aop-aspectj-annotation-example/">Spring AOP + AspectJ annotation example</a> <span><strong>(Spring 3.0)</strong></span><br> An example to show you how to integrate AspectJ annotation with Spring framework.</li> 
          <li><a href="http://www.mkyong.com/spring3/spring-aop-aspectj-in-xml-configuration-example/">Spring AOP + AspectJ in XML configuration example</a> <span><strong>(Spring 3.0)</strong></span><br> Spring AOP with AspectJ in XML base configuration.</li> 
         </ul> 
         <h2>Spring Object/XML Mapper (Spring 3.0)</h2> 
         <p>In Spring 3.0, Object to XML mapping (OXM) is moved from the Spring Web Services to the core Spring Framework.</p> 
         <ul> 
          <li><a href="http://www.mkyong.com/spring3/spring-objectxml-mapping-example/">Spring Object/XML mapping example</a><br> Spring oxm + castor, convert Object to XML and vice verse.</li> 
         </ul> 
         <p><a name="springjdbc"></a></p> 
         <h2>Spring JDBC Support</h2> 
         <p>Spring provides many helper classes to simplify the overall JDBC database operation.</p> 
         <ul> 
          <li><a href="http://www.mkyong.com/spring/maven-spring-jdbc-example/">Spring + JDBC example</a><br> An example to show how to integrate Spring and JDBC.</li> 
          <li><a href="http://www.mkyong.com/spring/spring-jdbctemplate-jdbcdaosupport-examples/">JdbcTemplate + JdbcDaoSupport example</a><br> Example to use Spring’s JdbcTemplate and JdbcDaoSupport classes to simplify the overall JDBC database operation processes.</li> 
          <li><a href="http://www.mkyong.com/spring/spring-jdbctemplate-querying-examples/">JdbcTemplate Querying examples</a><br> Here are a few examples to show how to use JdbcTemplate query() methods to query or extract data from database.</li> 
          <li><a href="http://www.mkyong.com/spring/spring-jdbctemplate-batchupdate-example/">JdbcTemplate batchUpdate() example</a><br> A batchUpdate() example to show how to perform the batch insert operations.</li> 
          <li><a href="http://www.mkyong.com/spring/spring-simplejdbctemplate-querying-examples/">SimpleJdbcTemplate Querying examples</a><br> More user friendly and simple way to query or extract data from database.</li> 
          <li><a href="http://www.mkyong.com/spring/spring-simplejdbctemplate-batchupdate-example/">SimpleJdbcTemplate batchUpdate() example</a><br> Another batch update example with SimpleJdbcTemplate, a java5-friendly supplement to JdbcTemplate.</li> 
          <li><a href="http://www.mkyong.com/spring/spring-named-parameters-examples-in-simplejdbctemplate/">Named Parameters examples in SimpleJdbcTemplate</a><br> An example to show how to use named parameter as SQL parameter values, and this is only supported in SimpleJdbcTemplate.</li> 
         </ul> 
         <h2>Spring Hibernate Support</h2> 
         <p>Spring comes with many handy classes to support Hibernate ORM framework.</p> 
         <ul> 
          <li><a href="http://www.mkyong.com/spring/maven-spring-hibernate-mysql-example/">Maven + Spring + Hibernate + MySql Example</a><br> A simple project of using Spring and Hibernate.</li> 
          <li><a href="http://www.mkyong.com/spring/maven-spring-hibernate-annotation-mysql-example/">Maven + (Spring + Hibernate) Annotation + MySql Example</a><br> A simple project of using Spring and Hibernate (annotation version).</li> 
          <li><a href="http://www.mkyong.com/spring/spring-aop-transaction-management-in-hibernate/">Spring AOP transaction management in Hibernate</a><br> An example to show how to manage the Hibernate transaction with Spring AOP. </li> 
          <li><a href="http://www.mkyong.com/struts/struts-spring-hibernate-integration-example/">Struts + Spring + Hibernate integration</a><br> Example to integrate Spring with Struts and Hibernate framework. </li> 
         </ul> 
         <h2>Spring E-mail Support</h2> 
         <p>Spring’s provides MailSender to send email via JavaMail API.</p> 
         <ul> 
          <li><a href="http://www.mkyong.com/spring/spring-sending-e-mail-via-gmail-smtp-server-with-mailsender/">Sending E-mail via MailSender</a><br> Example to use Spring’s MailSender to send an email via Gmail SMTP server.</li> 
          <li><a href="http://www.mkyong.com/spring/spring-define-an-e-mail-template-in-bean-configuration-file/">E-mail template in bean configuration file</a><br> Hard-code all the email properties and message content in the method body is not a good practice, you should consider define the email message template in the Spring’s bean configuration file.</li> 
          <li><a href="http://www.mkyong.com/spring/spring-sending-e-mail-with-attachment/">Sending e-mail with attachment</a><br> Example to use Spring to send an email that has attachments.</li> 
         </ul> 
         <h2>Spring Scheduling Support</h2> 
         <p>Spring has very good support in both JDK timer and Quartz framework.</p> 
         <ul> 
          <li><a href="http://www.mkyong.com/spring/spring-jdk-timer-scheduler-example/">Spring + JDK Timer scheduler example</a><br> An article about how Spring schedules a job with JDK timer.</li> 
          <li><a href="http://www.mkyong.com/spring/spring-quartz-scheduler-example/">Spring + Quartz scheduler example</a><br> An article about how Spring schedules a job with Quartz framework.</li> 
          <li><a href="http://www.mkyong.com/struts/struts-spring-quartz-scheduler-integration-example/">Spring + Struts + Quartz scheduler example</a><br> Integrate Spring with Struts and schedule a job with Quartz framework.</li> 
         </ul> 
         <h2>Integrating Spring with Other Web Frameworks</h2> 
         <p>Spring integrates with other web frameworks.</p> 
         <ul> 
          <li><a href="http://www.mkyong.com/spring/spring-how-to-do-dependency-injection-in-your-session-listener/">Spring dependency injection in the servlet session listener</a><br> Spring comes with a “ContextLoaderListener” listener as a generic way to enable the Spring dependency injection in session listener and almost all other web framework.</li> 
          <li><a href="http://www.mkyong.com/struts/struts-spring-integration-example/">Struts + Spring integration</a><br> Example to integrate Spring with Struts 1.x framework.</li> 
          <li><a href="http://www.mkyong.com/struts2/struts-2-spring-integration-example/">Struts 2 + Spring integration example</a><br> Example to integrate Spring with Struts 2 framework.</li> 
          <li><a href="http://www.mkyong.com/jsf2/jsf-2-0-spring-integration-example/">JSF 2.0 + Spring integration example</a><br> Example to integrate JSF 2.0 with Spring framework.</li> 
          <li><a href="http://www.mkyong.com/jsf2/jsf-2-0-spring-hibernate-integration-example/">JSF 2.0 + Spring + Hibernate integration example</a><br> Example to integrate JSF 2.0 + Spring + Hibernate framework together.</li> 
          <li><a href="http://www.mkyong.com/wicket/wicket-spring-integration-example/">Wicket + Spring integration example</a><br> Example to integrate Wicket with Spring framework.</li> 
          <li><a href="http://www.mkyong.com/struts2/struts-2-spring-quartz-scheduler-integration-example/">Struts 2 + Spring + Quartz scheduler integration example</a><br> Example to integrate Spring + Struts 2 + Quartz.</li> 
          <li><a href="http://www.mkyong.com/struts2/struts-2-spring-hibernate-integration-example/">Struts 2 + Spring + Hibernate integration example</a><br> Example to integrate Spring + Struts 2 + Hibernate.</li> 
         </ul>
