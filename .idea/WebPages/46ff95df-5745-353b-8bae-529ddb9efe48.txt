      <h2>About <a href="/questions/tagged/raspberry-pi" class="post-tag" title="show questions tagged 'raspberry-pi'" rel="tag">raspberry-pi</a></h2> 
      <div class="post-text"> 
       <div class="welovestackoverflow" id="wiki-excerpt"> 
        <p> The Raspberry Pi is a small, inexpensive ARM-based PC device. PROGRAMMING QUESTIONS ONLY. Questions about using it should be asked on https://raspberrypi.stackexchange.com/ </p> 
       </div> 
       <h2><a href="https://raspberrypi.stackexchange.com/">Raspberry Pi.SE</a></h2> 
       <p>There is a Stack Exchange <a href="https://raspberrypi.stackexchange.com/">site</a> dedicated to the Raspberry Pi. <strong>If your question is about using the Raspberry Pi, it should be asked there</strong>. However, general programming questions, which you happen to encounter on the Raspberry Pi are appropriate on Stack Overflow.</p> 
       <hr> 
       <p>The <a href="http://en.wikipedia.org/wiki/Raspberry_Pi" rel="nofollow noreferrer">Raspberry Pi</a> is a low cost, credit card sized, ARMv6 instruction set based PC device developed by the <a href="http://www.raspberrypi.org/" rel="nofollow noreferrer">Raspberry Pi foundation</a>. </p> 
       <p>The Raspberry Pi currently has seven models, <a href="https://www.modmypi.com/image/cache/data/raspberry-pi-model-a-hq3-small-800x800.jpg" rel="nofollow noreferrer">Model A</a> , <a href="http://images.bit-tech.net/content_images/2013/05/raspberry-pi-case-competition-voting/raspberry-pi-model-b-512mb-1280x1024.jpg" rel="nofollow noreferrer">Model B</a> , <a href="http://www.raspberrypi.org/products/model-a-plus/" rel="nofollow noreferrer">Model A+</a>, <a href="http://www.raspberrypi.org/products/model-b-plus/" rel="nofollow noreferrer">Model B+</a>, <a href="https://www.raspberrypi.org/products/raspberry-pi-2-model-b/" rel="nofollow noreferrer">Model 2 B</a>, <a href="https://www.raspberrypi.org/blog/raspberry-pi-zero/" rel="nofollow noreferrer">Zero</a> and <a href="https://www.raspberrypi.org/products/raspberry-pi-3-model-b/" rel="nofollow noreferrer">Model 3 B</a>. Model A was the first model of the Pi to be released (in February 2012), at an average price of $25. Model B was a higher spec variant (first released in June 2012) at an average price of $35. Currently, among the seven models, Model 3 B has the highest spec (released in February 2016) at an average price of $35. Model A+ is a low-cost variant (released in November 2014) at an average price of $20.</p> 
       <p>Model A consists of:</p> 
       <ul> 
        <li>ARM11 CPU</li> 
        <li>OpenGL ES 2.0 capable GPU</li> 
        <li>256 MB RAM (shared with GPU)</li> 
        <li>HDMI and RCA video</li> 
        <li>1x USB 2.0 port</li> 
        <li>3.5 mm audio jack</li> 
        <li>SD card slot</li> 
       </ul> 
       <p>Model B consists of:</p> 
       <ul> 
        <li>ARM11 CPU</li> 
        <li>OpenGL ES 2.0 capable GPU</li> 
        <li>512 MB RAM (shared with GPU)</li> 
        <li>HDMI and RCA video</li> 
        <li>2x USB 2.0 port</li> 
        <li>3.5 mm audio jack</li> 
        <li>Ethernet port</li> 
        <li>SD card slot</li> 
       </ul> 
       <p>Model A+ consists of:</p> 
       <ul> 
        <li>ARM11 CPU</li> 
        <li>OpenGL ES 2.0 capable GPU</li> 
        <li>256 MB RAM (shared with GPU)</li> 
        <li>HDMI and composite video via 3.5mm TRRS jack (shared with audio)</li> 
        <li>1x USB 2.0 port</li> 
        <li>3.5 mm audio jack</li> 
        <li>MicroSD card slot</li> 
       </ul> 
       <p>Model B+ consists of:</p> 
       <ul> 
        <li>ARM11 CPU</li> 
        <li>OpenGL ES 2.0 capable GPU</li> 
        <li>512 MB RAM (shared with GPU)</li> 
        <li>HDMI and composite video via 3.5mm TRRS jack (shared with audio)</li> 
        <li>4x USB 2.0 port</li> 
        <li>3.5 mm audio jack </li> 
        <li>Ethernet port</li> 
        <li>MicroSD card slot</li> 
       </ul> 
       <p>Model 2 B consists of:</p> 
       <ul> 
        <li>4 USB ports</li> 
        <li>40 GPIO pins</li> 
        <li>Full HDMI port</li> 
        <li>Ethernet port</li> 
        <li>Combined 3.5mm audio jack and composite video</li> 
        <li>Camera interface (CSI)</li> 
        <li>Display interface (DSI)</li> 
        <li>Micro SD card slot</li> 
        <li>VideoCore IV 3D graphics core</li> 
        <li>A 900MHz quad-core ARM Cortex-A7 CPU</li> 
        <li>1GB RAM</li> 
       </ul> 
       <p>Compared to the Raspberry Pi Model 2 B, Raspberry Pi Model 3 B has:</p> 
       <ul> 
        <li>A 1.2GHz 64-bit quad-core ARMv8 CPU</li> 
        <li>802.11n Wireless LAN</li> 
        <li>Bluetooth 4.1</li> 
        <li>Bluetooth Low Energy (BLE)</li> 
       </ul> 
       <p>The Raspberry Pi has no on-board operating system and requires an OS to be loaded onto an SD card, which it then boots from.</p> 
       <p>Common OS images</p> 
       <ul> 
        <li>NOOBS (New Out Of Box Software - ideal for first time users)</li> 
        <li>Raspbian (Stretch - suitable for beginners)</li> 
        <li>Ubuntu (Core or classic)</li> 
        <li>Soft-Float Debian</li> 
        <li>Pidora</li> 
       </ul>
