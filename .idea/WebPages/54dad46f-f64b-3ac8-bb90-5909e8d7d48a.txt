      <h2>About <a href="/questions/tagged/cgi" class="post-tag" title="show questions tagged 'cgi'" rel="tag">cgi</a></h2> 
      <div class="post-text"> 
       <div class="welovestackoverflow" id="wiki-excerpt"> 
        <p> 1) The Common Gateway Interface is a standard defining how web server software can delegate web page generation to a stand-alone application or executable file. 2) Computer Generated Imagery </p> 
       </div> 
       <p>This tag is mostly used for the <strong><a href="http://en.wikipedia.org/wiki/Common_Gateway_Interface" rel="nofollow">Common Gateway Interface</a></strong> (CGI), which is a standard (<a href="http://www.ietf.org/rfc/rfc3875" rel="nofollow">see RFC 3875: CGI Version 1.1</a>) that defines how web server software can delegate the generation of web pages to a stand-alone application, an executable file. Such applications are known as CGI scripts; they can be written in any programming language, although scripting languages are often used.</p> 
       <p>Less commonly on Stack Overflow, CGI may also be used to tag questions regarding <strong>Computer Generated Imagery</strong>.</p> 
       <p><strong>See also</strong></p> 
       <ul> 
        <li><a href="http://stackoverflow.com/tags/fastcgi/info">fastcgi</a></li> 
        <li><a href="http://stackoverflow.com/tags/fcgi/info">fcgi</a></li> 
        <li><a href="http://en.wikipedia.org/wiki/Common_Gateway_Interface" rel="nofollow">Wiki</a></li> 
        <li><a href="http://www.htmlhelp.com/faq/cgifaq.1.html" rel="nofollow">CGI Programming FAQ: Basic Questions</a></li> 
       </ul>
