        <h1>Java XML Tutorial</h1> 
        <div class="post-meta"> 
         <div class="post-meta">
          By 
          <a href="http://www.mkyong.com/author/mkyong/" title="mkyong class=" author" rel="author">mkyong</a> | 
          <time datetime="2010-04-02T09:59:36+00:00">April 2, 2010</time> | Updated : August 4, 2011 | Viewed : 821,640 |

        <div id="post-content"> 
         <div class="pic"> 
          <img src="http://www.mkyong.com/wp-content/uploads/2010/04/java-xml-tutorials.png" alt="java xml tutorials" title="java xml tutorials">

         <p>In Java JDK, two built-in XML parsers are available – <strong>DOM</strong> and <strong>SAX</strong>, both have their pros and cons. Here’s few examples to show how to create, modify and read a XML file with Java <strong>DOM</strong>, <strong>SAX</strong>, <strong>JDOM</strong>. </p> 
         <p>In addition, updated <strong>JAXB</strong> example to show you how to convert object to / from XML.</p> 
         <h2>DOM XML Parser</h2> 
         <p>The DOM is the easiest to use Java XML Parser. It parses an entire XML document and load it into memory, modeling it with Object for easy nodel traversal. DOM Parser is slow and consume a lot memory if it load a XML document which contains a lot of data. </p> 
         <ul> 
          <li><a href="http://www.mkyong.com/java/how-to-read-xml-file-in-java-dom-parser/">Read a XML file</a><br> Read a XML file and print it out each elements.</li> 
          <li><a href="http://www.mkyong.com/java/how-to-modify-xml-file-in-java-dom-parser/">Modify existing XML file</a><br> Modify an existing XML file, by update the element and attribute, and also how to delete a element.</li> 
          <li><a href="http://www.mkyong.com/java/how-to-create-xml-file-in-java-dom/">Create a new XML file</a><br> Create a XML file with new document, element and attribute.</li> 
          <li><a href="http://www.mkyong.com/java/how-to-count-xml-elements-in-java-dom-parser/">Count XML Elements</a><br> Search and count total number of elements in a XML file.</li> 
         </ul>

         <h2>SAX XML Parser</h2> 
         <p>SAX parser is work differently with DOM parser, it does not load any XML document into memory and create some object representation of the XML document. Instead, the SAX parser use callback function (<code>org.xml.sax.helpers.DefaultHandler</code>) to informs clients of the XML document structure. </p> 
         <ul> 
          <li><a href="http://www.mkyong.com/java/how-to-read-xml-file-in-java-sax-parser/">Read a XML file</a><br> Read a XML file via SAX callback methods.</li> 
          <li><a href="http://www.mkyong.com/java/how-to-read-utf-8-xml-file-in-java-sax-parser/">Read a UTF-8 XML file</a><br> Read a Unicode XML file via SAX callback methods.</li> 
          <li><a href="http://www.mkyong.com/java/sax-error-malformedbytesequenceexception-invalid-byte-1-of-1-byte-utf-8-sequence/">SAX Error – Invalid byte 1 of 1-byte UTF-8 sequence</a><br> Common SAX error for XML file which contains Unicode character.</li> 
          <li><a href="http://www.mkyong.com/java/sax-error-content-is-not-allowed-in-prolog/">SAX Error – Content is not allowed in prolog</a><br> Common SAX error for invalid XML file content.</li> 
         </ul> 
         <p><em>SAX Parser is faster and uses less memory than DOM parser.</em></p> 
         <script async src="//pagead2.googlesyndication.com/pagead/js/adsbygoogle.js"></script>

         <h2>JDOM XML Parser</h2> 
         <p>JDOM provides a way to represent that document for easy and efficient reading, manipulation, and writing. It’s an alternative to DOM and SAX.</p> 
         <ul> 
          <li><a href="http://www.mkyong.com/java/how-to-read-xml-file-in-java-jdom-example/">Read a XML file</a><br> Read a XML file and print it out each elements.</li> 
          <li><a href="http://www.mkyong.com/java/how-to-modify-xml-file-in-java-jdom/">Modify existing XML file</a><br> Modify an existing XML file, by update the element and attribute, and also how to delete a element.</li> 
          <li><a href="http://www.mkyong.com/java/how-to-create-xml-file-in-java-jdom-parser/">Create a new XML file</a><br> Create a XML file with new document, element and attribute.</li> 
         </ul> 
         <h2>JAXB Example</h2> 
         <p>JAXB, using annotation to convert Java object to / from XML file.</p> 
         <ul> 
          <li><a href="http://www.mkyong.com/java/jaxb-hello-world-example/">JAXB 2.0 hello world example</a><br> A detail example to show you use JAXB to do XML Marshalling (convert object to XML) and XML Unmarshalling (Convert XML to object).</li> 
         </ul> 
         <h2>XML &amp; Properties</h2> 
         <p>The <strong>java.util.Properties</strong> class has build-in functionality to convert properties file into XML file or vice versse.</p> 
         <ul> 
          <li><a href="http://www.mkyong.com/java/how-to-store-properties-into-xml-file/">Convert properties file into XML file</a></li> 
          <li><a href="http://www.mkyong.com/java/how-to-load-properties-from-xml-file/">Convert XML file into properties file</a></li> 
         </ul>
