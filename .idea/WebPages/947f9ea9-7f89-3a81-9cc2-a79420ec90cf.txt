        <h1>Java Regular Expression Tutorial</h1> 
        <div class="post-meta"> 
         <div class="post-meta">
          By 
          <a href="http://www.mkyong.com/author/mkyong/" title="mkyong class=" author" rel="author">mkyong</a> | 
          <time datetime="2010-04-07T04:11:39+00:00">April 7, 2010</time> | Updated : August 30, 2012 | Viewed : 164,522 | 
          <span>+145 pv/w</span>
         </div> 
        </div> 
        <div id="post-content"> 
         <div class="pic"> 
          <img src="http://www.mkyong.com/wp-content/uploads/2010/04/regular-expression-tutorials.png" alt="java regular expression tutorials" title="java regular expression tutorials"> 
         </div> 
         <p>Java has comprehensive support for Regular Expression functionality through the <strong>java.util.regex</strong> package. The regular expression language is easy to learn but hard to master, the better way to learn it is through examples. In theoretical, regular expression can match almost any stuff you want, the only limitation is in your imagination.</p> 
         <p>Happy learning Java Regular Expression :)</p> 
         <ul> 
          <li><a href="http://www.mkyong.com/regular-expressions/how-to-validate-username-with-regular-expression/" target="_blank">Username regular expression</a><br> Username regular expression example in Java and unit tested with TestNG.<p></p> <pre><code class="language-bash">
^[a-z0-9_-]{3,15}$
</code></pre> </li> 
          <li><a href="http://www.mkyong.com/regular-expressions/how-to-validate-password-with-regular-expression/" target="_blank">Password regular expression</a><br> Password regular expression example in Java and unit tested with TestNG.<p></p> <pre><code class="language-bash">
((?=.*\d)(?=.*[a-z])(?=.*[A-Z])(?=.*[@#$%]).{6,20})
</code></pre> </li> 
          <li><a href="http://www.mkyong.com/regular-expressions/how-to-validate-hex-color-code-with-regular-expression/" target="_blank">Hex color code regular expression</a><br> Hex color code regular expression example in Java and unit tested with TestNG.<p></p> <pre><code class="language-bash">
^#([A-Fa-f0-9]{6}|[A-Fa-f0-9]{3})$
</code></pre> </li> 
          <li><a href="http://www.mkyong.com/regular-expressions/how-to-validate-email-address-with-regular-expression/" target="_blank">E-mail address regular expression</a><br> E-mail address regular expression example in Java and unit tested with TestNG.<p></p> <pre><code class="language-bash">
^[_A-Za-z0-9-]+(\\.[_A-Za-z0-9-]+)*@
[A-Za-z0-9]+(\\.[A-Za-z0-9]+)*(\\.[A-Za-z]{2,})$
</code></pre> </li> 
          <li><a href="http://www.mkyong.com/regular-expressions/how-to-validate-image-file-extension-with-regular-expression/" target="_blank">Image file extension regular expression</a><br> Image file extension regular expression example in Java and unit tested with TestNG.<p></p> <pre><code class="language-bash">
([^\s]+(\.(?i)(jpg|png|gif|bmp))$)
</code></pre> </li> 
          <li><a href="http://www.mkyong.com/regular-expressions/how-to-validate-ip-address-with-regular-expression/" target="_blank">IP Address regular expression</a><br> IP Address regular expression example in Java and unit tested with TestNG.<p></p> <pre><code class="language-bash">
^([01]?\\d\\d?|2[0-4]\\d|25[0-5])\\.([01]?\\d\\d?|2[0-4]\\d|25[0-5])\\.
([01]?\\d\\d?|2[0-4]\\d|25[0-5])\\.([01]?\\d\\d?|2[0-4]\\d|25[0-5])$
</code></pre> </li> 
          <li><a href="http://www.mkyong.com/regular-expressions/how-to-validate-time-in-12-hours-format-with-regular-expression/" target="_blank">Time in 12 Hours format regular expression</a><br> Time in 12 Hours format regular expression example in Java and unit tested with TestNG.<p></p> <pre><code class="language-bash">
(1[012]|[1-9]):[0-5][0-9](\\s)?(?i)(am|pm)
</code></pre> </li> 
          <li><a href="http://www.mkyong.com/regular-expressions/how-to-validate-time-in-24-hours-format-with-regular-expression/" target="_blank">Time in 24 Hours format regular expression</a><br> Time in 24 Hours format regular expression example in Java and unit tested with TestNG.<p></p> <pre><code class="language-bash">
([01]?[0-9]|2[0-3]):[0-5][0-9]
</code></pre> </li> 
          <li><a href="http://www.mkyong.com/regular-expressions/how-to-validate-date-with-regular-expression/" target="_blank">Date regular expression</a><br> Date regular expression example in Java and unit tested with TestNG.<p></p> <pre><code class="language-bash">
(0?[1-9]|[12][0-9]|3[01])/(0?[1-9]|1[012])/((19|20)\\d\\d)
</code></pre> </li> 
          <li><a href="http://www.mkyong.com/regular-expressions/how-to-validate-html-tag-with-regular-expression/" target="_blank">HTML tag regular expression</a><br> HTML tag regular expression example in Java and unit tested with TestNG.<p></p> <pre><code class="language-bash">
&lt;("[^"]*"|'[^']*'|[^'"&gt;])*&gt;
</code></pre> </li> 
          <li><a href="http://www.mkyong.com/regular-expressions/how-to-extract-html-links-with-regular-expression/" target="_blank">HTML Links regular expression</a><br> HTML links regular expression example in Java and unit tested with TestNG.<p></p> <pre><code class="language-bash">
(?i)&lt;a([^&gt;]+)&gt;(.+?)&lt;/a&gt;
</code></pre> <pre><code class="language-bash">
\s*(?i)href\s*=\s*(\"([^"]*\")|'[^']*'|([^'"&gt;\s]+));
</code></pre> </li> 
         </ul>
