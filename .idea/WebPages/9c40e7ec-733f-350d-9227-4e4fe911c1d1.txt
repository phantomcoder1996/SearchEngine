        <h1>Java JSON Tutorial</h1> 
        <div class="post-meta"> 
         <div class="post-meta">
          By 
          <a href="http://www.mkyong.com/author/mkyong/" title="mkyong class=" author" rel="author">mkyong</a> | 
          <time datetime="2011-08-12T15:01:50+00:00">August 12, 2011</time> | Updated : August 23, 2011 | Viewed : 705,524 | 
          <span>+716 pv/w</span>
         </div> 
        </div> 
        <div id="post-content"> 
         <div class="pic"> 
          <img src="http://www.mkyong.com/wp-content/uploads/2011/08/java-json.png" alt="java json tutorials" title="java-json" width="250" height="100"> 
         </div> 
         <p><strong>JSON</strong> (JavaScript Object Notation), is a simple and easy to read and write data exchange format. It’s popular and implemented in countless projects worldwide, for those don’t like XML, JSON is a very good alternative solution.</p> 
         <p>In this series of Java JSON tutorials, we focus on three popular third party Java libraries to process JSON data, which are <strong>Jackson</strong>, <strong>Google Gson</strong> and <strong>JSON.simple</strong></p> 
         <h2>Jackson</h2> 
         <p>A High-performance JSON processor.</p> 
         <ul> 
          <li><a href="http://www.mkyong.com/java/how-to-convert-java-object-to-from-json-jackson/">Jackson data binding example – Object &amp; JSON</a><br> Example to use ObjectMapper writeValue() and readValue() to convert Java object to / from JSON.</li> 
          <li><a href="http://www.mkyong.com/java/how-to-convert-java-map-to-from-json-jackson/">Jackson data binding example – Map &amp; JSON</a><br> Example to convert Java Map to / from JSON</li> 
          <li><a href="http://www.mkyong.com/java/jackson-tree-model-example/">Jackson Tree Model example</a><br> Read JSON into Tree, like DOM for XML.</li> 
          <li><a href="http://www.mkyong.com/java/jackson-streaming-api-to-read-and-write-json/">Jackson Streaming example</a><br> Example to use JsonGenerator() and JsonParser() to stream read and write JSON data.</li> 
          <li><a href="http://www.mkyong.com/java/how-to-enable-pretty-print-json-output-jackson/">Pretty print JSON output in Jackson</a><br> Pretty print JSON output via defaultPrettyPrintingWriter().</li> 
         </ul>

         <h2>Google Gson</h2> 
         <p>It was originally created for use inside Google, now it is used by many <a href="http://sites.google.com/site/gson/gson-users" target="_blank">public projects</a>.</p> 
         <ul> 
          <li><a href="http://www.mkyong.com/java/how-do-convert-java-object-to-from-json-format-gson-api/">Gson data model example – Object &amp; JSON</a><br> Example to use toJson() and fromJson() to convert Java Object to / from JSON.</li> 
          <li><a href="http://www.mkyong.com/java/gson-streaming-to-read-and-write-json/">Gson Streaming example</a><br> Example to use JsonWriter() and JsonReader () to stream read and write JSON data.</li> 
          <li><a href="http://www.mkyong.com/java/how-to-enable-pretty-print-json-output-gson/">Pretty print JSON output in Gson</a><br> Pretty print JSON output via GsonBuilder.</li> 
         </ul>

         <h2>JSON.simple</h2> 
         <p>Simple Java library for JSON, to read and write JSON data.</p> 
         <ul> 
          <li><a href="http://www.mkyong.com/java/json-simple-example-read-and-write-json/">JSON.simple example – Read and write JSON </a><br> Example to show you how to read and write JSON data to / from file.</li> 
         </ul>
