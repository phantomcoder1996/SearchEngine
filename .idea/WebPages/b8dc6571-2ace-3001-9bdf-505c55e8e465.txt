        <h1>Oracle PL/SQL – CREATE function example</h1> 
        <div class="post-meta"> 
         <div class="post-meta">

          <a href="https://www.mkyong.com/author/dhaval/" title="Dhaval Dadhaniya class=" author" rel="author">Dhaval Dadhaniya</a> | 
          <time datetime="2017-08-30T07:22:52+00:00">August 30, 2017</time> | Viewed : 16,495 | 
          <span>+948 pv/w</span>
         </div> 
        </div> 
        <div id="post-content"> 
         <p>This article will help you to understand how to create a user defined function. It’s also known as stored function or user function.</p> 
         <ol> 
          <li>User defined functions are similar to procedures. The only difference is that function always returns a value.</li> 
          <li>User defined functions can be used as a part of an SQL expression.</li> 
         </ol> 
         <div class="note"> 
          <strong>Note</strong>
          <br> Oracle SQL does not support calling of functions with Boolean parameters or returns. 
         </div> 
         <h2>1. Function – Get formatted address</h2> 
         <p>In this example, we will create a function to get formatted address by giving the person’s name.</p> 
         <p>1.1 Create tables and function.</p> 
         <pre><code class="language-sql">
-- creating table person_info

CREATE TABLE person_info
(
  PERSON_ID number(5) primary key,
  FIRST_NAME varchar2(20),
  LAST_NAME varchar2(20)
);
</code></pre> 
         <pre><code class="language-sql">
--creating table person_address_details

CREATE TABLE person_address_details
(
  PERSON_ADDRESS_ID number(5) primary key,
  PERSON_ID number(5) references person_info(person_id),
  CITY varchar2(15),
  STATE varchar2(15),
  COUNTRY varchar2(20),
  ZIP_CODE varchar2(10)
);
</code></pre> 
         <pre><code class="language-sql">
--creating function get_complete_address

create or replace FUNCTION get_complete_address(in_person_id IN NUMBER) 
   RETURN VARCHAR2
   IS person_details VARCHAR2(130);

   BEGIN 

	  SELECT 'Name-'||person.first_name||' '|| person.last_name||', 
        City-'|| address.city ||', State-'||address.state||', 
        Country-'||address.country||', ZIP Code-'||address.zip_code 
      INTO person_details
      FROM person_info person, person_address_details address
      WHERE person.person_id = in_person_id 
      AND address.person_id = person.person_id;

      RETURN(person_details); 

    END get_complete_address;
</code></pre>

         <h2>2. Function – Check Palindrome String</h2> 
         <p>In this example, we will create a function to check whether a given string is palindrome or not.</p> 
         <blockquote>
          <p>A palindrome is a word, phrase, number, or other sequence of characters which reads the same backward as forward, such as madam or racecar.<br> <a href="https://en.wikipedia.org/wiki/Palindrome" target="_blank">https://en.wikipedia.org/wiki/Palindrome</a></p>
         </blockquote> 
         <p>2.1 Creating the function.</p> 
         <pre><code class="language-sql">
CREATE OR REPLACE FUNCTION checkForPalindrome(inputString VARCHAR2)
   RETURN VARCHAR2 
   IS result VARCHAR2(75);
   
   reversedString VARCHAR2(50); 
   BEGIN 
      SELECT REVERSE(inputString) INTO reversedString FROM DUAL;
            
      -- Using UPPER to ignore case sensitivity.
      IF UPPER(inputString) = UPPER(reversedString)
      THEN
      RETURN(inputString||' IS a palindrome.');
      END IF;
      RETURN (inputString||' IS NOT a palindrome.');
      
    END checkForPalindrome;
/
</code></pre> 
         <p>2.2 Calling the function.</p> 
         <pre><code class="language-sql">
SELECT checkForPalindrome('COMPUTER') FROM DUAL;
--	Output
-- 	COMPUTER IS NOT a palindrome.


SELECT checkForPalindrome('MAdam') FROM DUAL;
--	Output
-- 	MAdam IS a palindrome.


SELECT checkForPalindrome('KANAK') FROM DUAL;

--	Output
-- 	KANAK IS a palindrome.
</code></pre> 
         <h2>3. Function – Calculate income tax</h2> 
         <p>In this example, we will create a function to calculate income tax, assumed tax rate is 30% of all annual income from salary. </p> 
         <p>3.1 Create tables and function.</p> 
         <pre><code class="language-sql">
--creating table person

CREATE TABLE person
(
  PERSON_ID number(5) primary key,
  FULLNAME varchar2(20)
);
</code></pre> 
         <pre><code class="language-sql">
--creating table person_salary_details

CREATE TABLE person_salary_details
(
  SALARY_ID number(5) primary key,
  PERSON_ID number(5) references person(person_id),
  SALARY number(8),
  MONTH varchar2(9),
  YEAR number(4)
);
</code></pre> 
         <pre><code class="language-sql">
--creating function

CREATE OR REPLACE FUNCTION calculate_tax(personId NUMBER)
   RETURN NUMBER
   IS tax NUMBER(10,2);

BEGIN 
   tax := 0;
   
      SELECT (sum(salary)*30)/100 INTO tax FROM person_salary_details WHERE person_id = personId;
            
      RETURN tax;
      
END calculate_tax;
</code></pre>
