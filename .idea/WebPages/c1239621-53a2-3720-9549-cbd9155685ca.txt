           <div class="post-text" itemprop="text"> 
            <p>I think the issue seems to with your expectations to see the browser. You are launching the browser inside a docker container. The docker container doesn't have a UI, the browser is launched inside the container instance and not on your machine which initiates the call</p> 
            <p>The default <code>selenium/standalone-firefox</code> doesn't allow viewing of the browser. If you need to view the browser then you need to use the <code>selenium/standalone-firefox-debug</code> image. This image hosts VNC at port <code>5900</code> of the container. You will run it using below</p> 
            <pre><code>docker run -d -p 4445:4444 -p 5900:5900 selenium/standalone-firefox-debug
</code></pre> 
            <p>Then you will execute your code and connect to <code>127.0.0.1:5900</code> using a VNC Viewer. It will ask for a password which is <code>secret</code> by default.</p> 
            <p>Then you will see the browser in that VNC</p> 
            <p><a href="https://i.stack.imgur.com/6FFrS.png" rel="nofollow noreferrer"><img src="https://i.stack.imgur.com/6FFrS.png" alt="VNC View"></a></p> 
            <p>The above is the result of running the code you had in question</p>
