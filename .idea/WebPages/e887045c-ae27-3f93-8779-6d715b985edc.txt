                <p>There are three part of ternary operator:<br> (Condition)?truepart:falsepart</p> 
                <p>1. Condition- any boolean expression<br> 2. True part- if condition evalute true so assign this value left hand side.<br> 3. False part- if condition evalute false so assign this value left hand side.</p>
